Arduino Zoo Program
What it does
• 	The code stores a list of animals and the zoo location they belong to.
• 	When you type an animal name in the Serial Monitor, the program checks the list and tells you which zoo that animal is in.
• 	If the name doesn’t match, it says the animal is in a secret location.
How to use
1. 	Upload the code to your Arduino.
2. 	Open the Serial Monitor (set baud rate to 9600).
3. 	Type one of these names: , , , or .
4. 	The Arduino will reply with the zoo location.
